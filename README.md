# oracle-weekly-forecast

個人化占星週報 2026（十二宮位版），對應使用者的太陽、上升與生命靈數，產生每一週的十二宮運勢、幸運元素與每日提醒。
https://stella98052.github.io/oracle-weekly-forecast/

## 功能特色

- 使用者輸入基本出生資料（生日、時間、城市與時區），前端以 Placidus 宮位制計算上升星座（演算法參考 astro.com 的設定）。
- 依照太陽星座搭配每週模式，產生十二宮星等評分與具體行動建議。
- 顯示本週幸運色、幸運數字、幸運日與關鍵詞。
- 列出一週七天的簡短每日運勢與月亮所在星座。
- 支援中英文切換，全部計算都在瀏覽器端完成，無伺服器與後端依賴。

## 開發與部署

這個專案是一個純前端的單一 `index.html`，只要放在 GitHub Pages 或任何靜態網頁空間即可運作。

建議部屬方式：

1. 將 `index.html` 直接放在 repo 根目錄。
2. 在 GitHub 專案設定中啟用 Pages，Source 選擇 `main` 分支、根目錄。
3. 等待 GitHub 建立完成後，就可以透過 Pages 網址開啟使用。
